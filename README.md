# Sand-dancer, redux

Recreating Aaron Reed and Alexei Othenin-Girard's Sand-dancer in [Twine](https://twinery.org/). Written with [Harlowe](https://twine2.neocities.org/) (v.3.3.9).

## The original

[*Sand-dancer*](https://sand-dancer.textories.com/) was created to illustrate [Aaron Reed](https://aaronareed.net/)'s [*Creating Interactive Fiction with Inform 7*](https://inform7.textories.com/).

*Creating Interactive Fiction with Inform 7* was first published in 2010 and is sadly out of print, but Aaron's [companion site](https://inform7.textories.com/sand-dancer/) and the source code are still available. You can play the finished game at [sand-dancer.textories.com](https://sand-dancer.textories.com/)

## Why in Twine

Twine excels at making [Choose Your Own Adventure](https://en.wikipedia.org/wiki/Choose_Your_Own_Adventure) style games, where the player/reader makes choices by clicking links. Inform is a great tool for making parser-based games where the player types what they want their character to do. While this makes for great puzzle games, it doesn't lend itself to mobile use. A friend introduced me to Twine and I made a text adventure version of my [2025 Christmas Scavenger hunt](https://www.heithcock.com/scavengerhunts/2025-christmas-page-turner) with it and was pleased with how easy it was to create and [the result](https://www.44westwind.com/page-turner).

I happened to reread my old copy of *Creating Interactive Fiction with Inform 7* and thought that it *really* wouldn't be that difficult to port the game to Twine and so here we are....

## To get started

You can play the Twine version of *Sand-dancer* at [44westwind.com/sand-dancer](https://www.44westwind.com/sand-dancer). To see this in Twine, clone this repository or just download the sand-dancer.twee file. Go to twinery.org, choose "Use in your browser", step through or skip the intro, then choose *Library* from the top menu, and then *Import*. Select the sand-dancer.twee file, double click the new "Sand Dancer" card. Choose *Help* from the top-right menu if this is your first time using Twine.
